Barang Barang — HDB Barter App
A furniture exchange platform for HDB residents
Civic Tech · Community

Disposing of bulky furniture is a real pain point for HDB residents — costly, inconvenient, and wasteful. I conceived and designed a peer-to-peer exchange platform that lets neighbours list, discover, and swap furniture, turning unwanted items into community resources.

Ideator, UX Researcher & Designer
Figma, Maze
User Research, Persona, Scenerio, B/C Testing
Web App
UX process
Discover
User research & problem framing
Identified the problem through observation and conversations with HDB residents. Mapped pain points around bulky item disposal — cost, inconvenience, and lack of community channels for reuse.
Interviews · Affinity mapping
Define
Personas & journey mapping
Synthesised research into two primary personas — the "downsizer" letting go of large items and the "first-time resident" furnishing on a budget. Mapped their end-to-end journeys to surface key friction points and opportunity areas.
Personas · Journey maps
Ideate
Concept development & wireframing
Originated the barter exchange concept — going beyond a simple marketplace to enable direct item-for-item swaps. Sketched low-fidelity flows then built mid-fidelity wireframes in Figma, exploring listing, browsing, and negotiation patterns.
Figma · Wireframes · User flows
Test
B/C testing via Maze
Ran structured B/C tests using Maze to compare two variants of the listing flow and the swap confirmation screen. Analysed task completion rates, misclick rates, and time-on-task to determine which design pattern was clearer to users before finalising the UI.
Maze · B/C testing · Usability metrics
Deliver
High-fidelity prototype & handoff
Produced a high-fidelity Figma prototype incorporating test findings. Documented component decisions and interaction patterns for developer handoff.
Hi-fi prototype · Design system · Handoff
Key outcomes outcomes
Validated through Maze testing Community-first UX Circular economy Reduced household waste

Heritage Explorer — Full-stack App
Route planning, gamified badges & a digital heritage passport
Full-stack · Gamification

A full-stack web app that transforms Singapore's heritage trails into an interactive, gamified journey. Users plan routes on a live map, complete on-site activities, upload photos, earn heritage badges, and track their visits in a personal digital passport — making cultural exploration tangible and memorable.

UX process
Ideate
Concept & gamification framework
Originated the idea of a heritage passport — applying gamification principles (badges, progress tracking, achievements) to encourage deeper engagement with Singapore's cultural sites beyond passive tourism.
Concept ideation · Gamification design
Design
UI design & prototyping in Figma (above carousel)
Designed the full interface in Figma — including the map view, site detail pages, activity upload flow, badge collection screen, and passport layout. Built interactive prototypes to validate navigation logic and onboarding flow before development.
Figma · Interactive prototype · UI system
Build
Frontend & backend development in Vue.js
Implemented the full application as both UI frontend and backend developer using Vue.js. Built the interactive map, photo upload pipeline, badge logic, and passport data model — bridging design intent with working code end-to-end.
Vue.js · Frontend · Backend · API integration
Deliver
Live deployment & iteration
Shipped the app to a live environment. Gathered feedback from initial users and iterated on the badge earn flow and passport UI based on real usage patterns.
Live deploy · Feedback loops · Iteration
